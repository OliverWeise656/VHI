Fragebogen Stimme und Hals

Bitte kennzeichnen Sie die Häufigkeit und den Schweregrad der folgenden Symptome, indem Sie die entsprechenden Zahlen ankreuzen und senden Sie den Fragebogen ab.

Danach werden Sie zur Stimmanalyse weitergeleitet.

  • Häufigkeit: 0 = nie, 1 = selten, 2 = manchmal, 3 = oft, 4 = immer
  • Schweregrad: 0 = nicht vorhanden, 1 = leicht, 2 = mittel, 3 = schwer, 4 = sehr schwer
